# Multi-Area-OSPF-Laboratory-in-GNS3
Настройке, диагностике и устранению неисправностей в многоуровневой сети OSPF.
## `Описание` 
Данный репозиторий содержит полную лабораторную работу по настройке Multi-Area OSPF в эмуляторе GNS3.
Полные конфигурации всех роутеров
Проверки работоспособности
Типовыенеисправности с методами диагностики и устранения
Практические сценарии для отработки навыков troubleshooting
## Цели лабораторной работы
Понять принципы иерархического дизайна OSPF
Настроить OSPF сеть с различными типами зон
Научиться диагностировать типовые неисправности
Освоить методы устранения проблем на разных уровнях модели OSI
Анализировать таблицы маршрутизации и типы LSA
